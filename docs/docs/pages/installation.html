Installation Instuctions


Installation Process

Installing Ness PHP is very easy. You can select one of the installation methods and get ready with in 3 mins. Every installation method creates same files in your project directory so you can select the best method fits you. Here is a list of methods you can use;

  1. Install using Composer
  2. Install by Downloading from GitHub Releases
  3. Install by Cloning the project from GitHub


Install Using Composer

You can use 'composer' package manager for downloading and installing the framework.

If you are on Windows you can use SHIFT + Right Click on your development folder(development folder; can be htdocs[for XAMP], www, public etc), and click Open Command Line Here 
composer create-project nessphp/project  project_name
project_name can be replaced with your own project name like; SimpleBlog, WebMessenger, mysimpleapplication, nessapplication etc...


Install by Downloading Package

If you do not have composer installed on your machine you can use github directly to create a project with Ness PHP. Here is the required steps for creating a project using GitHub Releases.

  1. Visit Application Repository / Releases page
  2. Download last version of the framework
    1. [Note: Project repository contains only stable releases of the framework. You can follow the same process with framework repository for creating a project from last unstable/development branch]
  3. Unzip the package you have downloaded.
  4. Optional: Rename the package/project name you have downloaded with your own project name
  5. Copy/Upload your project to local/remote server
That's All. Now you are ready to create amazing apps with your freshly installed Ness PHP Framework

Please DO NOT FORGET You will need to edit app_config file in project package correctly to make your new Ness PHP application run. Do not panic configuration will take less then 1 min. Just read Project Structure and Project Configuration topics before.


Server Requirements

Ness Framework is a lightweight, easy-to-code and adaptable library. You can be confident that we will consider the 'lightness' principle in all our releases. The only requirements are;

  1. PHP Version: =>5.6
  2. Extension: PHP PDO for database operations.
  3. Extension: PHP SimpleXml for Resource Management Class
